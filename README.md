> 🎤 : 만든 이유가 뭔가요? 

호텔매니저와 국가통역안내사를 토대로 관광통역업무를 겸하며 많은 외국인들이 생각보다 많은 외국인들이 잘못된 정보를 접하는걸 보게 되었습니다.
놀기도 부족한 시간과 아까운 돈을 낭비하는 그들의 고충을 덜고 자랑스런 대한민국의 서울을 알리고자 웹 사이트를 개발했습니다.


> 🎤 : 웹은 영문인데 Read Me 는 한국어군요?
